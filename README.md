# XUFAINOVA - Política de Privacidad y Condiciones del Servicio

**Última actualización:** 26 de marzo de 2026

---

## 1. Política de Privacidad

**Información que recopilamos:**
- Nombre de usuario y tag de Discord
- ID de usuario de Discord
- Contenido de mensajes enviados en canales específicos (ej. sugerencias)

**Cómo usamos la información:**
- Procesar sugerencias y enviar resultados a canales correspondientes
- Monitorizar bots y notificar desconexiones en canales de administración
- Los datos **no se venden ni comparten con terceros**

**Seguridad de los datos:**
- Los datos se almacenan temporalmente solo para el funcionamiento del bot
- No se usan para publicidad ni se envían fuera de Discord salvo las funciones descritas

**Compartir información con terceros:**
- No se comparte información personal con terceros
- Solo administradores del servidor y el bot pueden ver los datos procesados

**Cambios en la política:**
- Podemos actualizar esta política en cualquier momento
- Cambios importantes serán notificados a los usuarios

**Contacto:**
- Discord: `Korvilash`

## 2. Condiciones del Servicio

**Aceptación de las condiciones:**
- Al usar XUFAINOVA, aceptas estas condiciones. Si no estás de acuerdo, no uses el bot.

**Uso permitido:**
- Usar solo en servidores donde tengas permiso de administradores
- No usar para spam, acoso o infringir leyes
- Usar las funciones del bot según su finalidad (sugerencias, monitorización, roles, etc.)

**Responsabilidades del usuario:**
- Eres responsable de los mensajes que envías usando el bot
- Mantener seguros tus permisos y tokens de Discord

**Limitaciones:**
- XUFAINOVA no se responsabiliza por pérdidas de datos o errores de Discord
- No se garantiza disponibilidad continua o sin errores

**Modificaciones:**
- Nos reservamos el derecho de modificar estas condiciones en cualquier momento
- Los cambios son efectivos al publicarse; revisa regularmente
