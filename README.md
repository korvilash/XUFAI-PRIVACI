# XUFAI-PRIVACI

Política de Privacidad - Xufa Nova
Última actualización: 26 de marzo de 2026

1. Información que recopilamos
XufaiNova no recopila información personal de los usuarios salvo la estrictamente necesaria para funcionar:

Nombre de usuario y tag de Discord
ID de usuario de Discord
Contenido de mensajes enviados en canales específicos (por ejemplo, sugerencias)
2. Cómo usamos la información
La información se utiliza únicamente para las funciones del bot:

Procesar sugerencias y enviar resultados a canales correspondientes
Monitorizar bots y notificar desconexiones en canales de administración
Los datos nunca se venden ni se comparten con terceros.

3. Seguridad de los datos
Los datos se almacenan temporalmente solo para el funcionamiento del bot.
No se utilizan para publicidad ni se envían fuera de Discord salvo por las funciones descritas.
4. Compartir información con terceros
No compartimos información personal de los usuarios con terceros. Solo los administradores del servidor y el bot tienen acceso a la información procesada para sugerencias o monitorización de bots.

5. Cambios en la política
Nos reservamos el derecho de actualizar esta política en cualquier momento. Los cambios importantes serán notificados a los usuarios.

6. Contacto
Si tienes dudas sobre cómo manejamos tus datos, puedes contactarnos en Discord: Korvilash.
